# Glossary

# Console

Command line interface to actively input code.

# CSV File

A comma-separated values file that stores structured album data. It is the primary backend data source for the application.

# Dataset

The collection of albums, artists, bands, and other information providing the functions for the UI interface loaded on Rstudio.

# Data Frame

R data structure used for storing tabular data.

# Deployment

The publishing of the application onto the live Shiny hosting environment.

# dplyr

An R package used for data manipulation required to run MyFavoriteAlbums.

# ggplot2

An R package used for data visualization such as the graph to compare artists

# R

An open source programming language and software environment used for statistical computing.

# RStudio

An integrated development environment (IDE) for developing and running R code.

# shiny

An R package used to build and run code on a UI.

# Shiny Apps

Web application to publish and visualize code from R language
